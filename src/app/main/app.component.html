Demo: WhatsApp and SMS Cross Conversation

This example shows how Project Wingman could possibly work. It is built on the Twilio Conversations API and is used to demonstrate cross-channel communication. In the future, a user who doesn't have access to the Internet in a developing country could use a cheap WhatsApp plan to communicate using SMS with most of the fees going to the person who is configuring this using Project Wingman.

A user could also use this system (obviously if it was more cleaned up) to use WhatsApp and SMS at the same time (or whatever is cheapest) to combine 2 apps to 1 conversation.
